Laravel - Custom Validation Rules

Laravel 5.5 introduced the support for custom validation rules, so now we can create validation rules for forms easily. let's look at a complete example of validating a form and displaying the error messages back to the user.

Creating Rule Class.

Run below command using your terminal. 


                                
                                // Before
                                @if ($errors->has('email'))
                                    {{ $errors->first('email') }}
                                @endif
                                
                                // After:
                                @error('email')
                                    {{ $message }}
                                @enderror

Now run your application on your browser. 


                                php artisan serve

Wrapping Up

Weldone fellas, Now you know how to create a custom rule in laravel. but always try to use default rules, Happy Coding.
